# 使いかた

Refined Itsukara.link の使い方を説明します。チュートリアルをお探しの場合は、もう一度表示することができます。

<TutorialButton />

## 配信

![配信のスクリーンショット](timetable.png)

[配信ページ](/)からは、過去の配信と、これから始まる配信です、各配信をタップすると YouTube の配信画面が表示されます。左右にスクロールすると時間を前後に移動させることが出来ます。

マウスをお使いの方は <kbd>Shift</kbd> を押しながらホイールを回転させることで水平方向にスクロールできます。
